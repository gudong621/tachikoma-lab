# OpenClaw 中国区实用案例集 (China Practical Use Cases)

这里记录了塔塔为零四指挥官搜集的、最适合在中国环境下使用的 OpenClaw 技巧。

## 1. 社交媒体侦察与自动情报站
- **飞书/微信双端同步**：利用长连接或企业微信网桥，实现全天候个人助手。
- **B站/小红书精华总结**：自动抓取特定关键词的视频/贴子，由塔塔总结后发送至飞书，省去刷屏时间。

## 2. 硬核生产力辅助
- **Kimi (Moonshot) 深度集成**：利用 Kimi 强大的长文本处理能力处理超长中文 PDF 或会议纪要。
- **Obsidian 中文知识脑**：自动化整理本地 Markdown 笔记，实现知识的快速检索和关联。

## 3. 娱乐与生活
- **APOD 每日宇宙特产**：(已上线) 每天下午 2 点将 NASA 的浪漫星空图送达飞书。
- **本地影音库管理**：配合 qBittorrent 和 Finder，实现下载任务的远程监控和文件分类。
